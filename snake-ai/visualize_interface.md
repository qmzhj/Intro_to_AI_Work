# 训练过程可视化回放界面说明

## 概述

将多个算法的训练过程以**并排对比**的方式回放，展示 AI 从"笨"到"聪明"的完整学习历程。支持交互式窗口和视频导出两种模式。

### 分区说明

| 区域 | 位置 | 内容 |
|------|------|------|
| **顶栏** | 顶部 36px | 播放状态、速度、当前阶段/总帧数 |
| **标题** | 每列顶部 | 算法名称、当前回放的训练轮次(ep) |
| **游戏画面** | 每列中部 | 12×12 贪吃蛇棋盘，左上角显示分数 HUD |
| **进度条** | 游戏画面下方 | 当前录制文件的播放进度（颜色与算法对应） |
| **状态信息** | 进度条下方 | 当前帧/总帧数 + 当前奖励值 |
| **个体曲线** | 每列下方 | 该算法的训练奖励曲线（随着阶段推进逐步延伸） |
| **总对比** | 底部 180px | 所有算法完整训练曲线的平滑对比 |

## 游戏画面视觉元素

| 元素 | 说明 |
|------|------|
| **蛇身** | 渐变色（深蓝绿 → 亮绿），越靠近头部越亮 |
| **蛇头** | 亮绿色 (`rgb(50, 255, 50)`) |
| **食物** | 红色方块，带边框 |
| **网格** | 深灰色线条分隔 |
| **动作热力图** | 蛇头周围四个方向显示半透明色块 — 红色（高价值）→ 蓝色（低价值），反映 DQN 的 Q 值或 PPO 的动作概率 |
| **分数 HUD** | 左上角黄色数字，半透明黑底 |

## 交互式窗口（无 `--output`）

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `=` / `+` | 加速（1x → 2x → 4x → 8x → 16x） |
| `-` | 减速 |
| `→` | 暂停时下一帧 |
| `←` | 暂停时上一帧 |
| `r` | 重置到开头并自动播放 |
| `Esc` | 退出 |

## 视频导出

```bash
python visualize.py experiments/dqn_xxx experiments/ppo_xxx --output demo.mp4
```

- 使用 `SDL_VIDEODRIVER=dummy` 无头模式，不弹窗
- 通过 OpenCV (`cv2.VideoWriter`) 逐帧写入 MP4
- 帧率 10fps
- 导出时自动渲染所有帧并显示进度

## 阶段推进机制

- 每个实验目录包含若干 `.npz` 录制文件（由训练时每 `RECORD_INTERVAL` 回合保存）
- 录制文件按训练轮次编号排序，依次播放
- 每当切换到一个新的录制阶段：
  - 游戏画面切换到该阶段的 checkpoint 回放
  - 个体训练曲线向前延伸到该阶段对应的轮次
  - 曲线上的竖线标记当前阶段位置
- 所有算法在时间轴上**对齐播放**（同一全局帧序号），各自展示对应阶段的回放

## 训练曲线说明

- **个体曲线**：每个算法列中下方的小图，只绘制到当前阶段对应的 episode。半透明细线为原始奖励，彩色粗线为平滑后曲线（窗口 50），红色竖线标记当前位置。
- **总对比曲线**：底部固定显示所有算法完整训练过程的平滑后曲线对比，用于宏观评估各算法收敛速度和最终性能。
